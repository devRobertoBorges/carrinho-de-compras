# 🛒 Carrinho de Compras em JavaScript

Projeto simples de carrinho de compras no terminal, desenvolvido em JavaScript com foco em praticar lógica de programação, manipulação de arrays e modularização de código.

---

## 📌 Funcionalidades

- ➕ Adicionar itens ao carrinho
- ➖ Remover quantidade específica de um item
- ❌ Excluir item do carrinho
- 💰 Calcular valor total da compra
- 👀 Visualizar todos os itens adicionados

---

## 🧠 Conceitos praticados

Este projeto foi criado para reforçar conceitos como:

- Funções assíncronas (`async/await`)
- Arrays e métodos nativos:
  - `push()`
  - `splice()`
  - `findIndex()`
  - `reduce()`
  - `forEach()`
- Modularização com ES Modules
- Validação de dados
- Boas práticas de organização
